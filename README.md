# agrinho2026
projeto agrinho 
link do projeto: https://yasmimtisseusilva-rgb.github.io/agrinho2026/

Tema: agro forte, futuro verde


objetivo:O grande propósito é demonstrar que a sustentabilidade se tornou o principal ativo econômico do setor. Práticas verdes não apenas protegem o planeta, mas também abrem portas para mercados internacionais exigentes, geram créditos de carbono e garantem a viabilidade do agronegócio para as próximas gerações.


Fontes:EMBRAPA (Empresa Brasileira de Pesquisa Agropecuária),
 Ministério da Agricultura e Pecuária (MAPA), 
IBGE (Instituto Brasileiro de Geografia e Estatística).
